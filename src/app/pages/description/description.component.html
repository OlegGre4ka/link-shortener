Прописываю скрипт сокращения ссылки: обрабаттываю полученные данные, складываю в обЪект links, объекты с помощью метода push() определяем в массив, и выводим данные из массива в шабоне компонента linksshort.html с помощью директивы цикла *ngFor и интерполяции- {{}.}.
    Создаю отдельный сервис для отправки данных формы сокращения ссылки на сервер, прописываю POST-method, инжекчу данный сервис в linksshort.component и вуаля, данные полетели на серверок)!

Создаю роутинг-модуль для регистрации роутов в данном модуле,использую массив children для того чтобы прописать дочерние роуты. Создаю формы для регистрации и входа в аккаунт, прописываю валидацию с помощью Reactive-Driven подхода. Устанавливаю в проект json-server , тем самым имитирую работу сервера у себя на локалхосте. Пользуясь этим сервером провожу CRUD - операци по отправке на сервер и проверке введенных данных.Создаю асинхронный валидатор для для динамической проверки емейла на наличие в базе Асинхронный валидатор должен возвращать либо Observable либо Promise типа NULL если всё удачно и типа ValidationErrors если валидация не прошла( https://angular.io/api/forms/AsyncValidatorFn) На эту работу у меня ушло 2,5 дня (включая вёрстку).
